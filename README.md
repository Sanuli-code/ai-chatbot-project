# AI-Powered Customer Support Chatbot

This repository contains the academic project for developing an AI-powered customer support chatbot using a Python-based NLP model.

## Structure

- `src/` – Python source files (NLP model, FastAPI backend, etc.)
- `data/` – Training data, intents, and sample conversations.
- `docs/` – Requirements, architecture diagrams, design notes.

## Purpose

This repository is used to demonstrate configuration management and version control practices using GitHub as part of the project assessment.
